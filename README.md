# NTAP with CEA markov model

## NTAP Application Package
Full materials for New Technology Add-on Payment submission to CMS, inspired by Yuan et al. (2020) on XGBoost for early sepsis diagnosis in ICUs.

- [Introduction & Overview](01-Introduction_and_Overview.pdf) — Study citation, synthetic data link, CEA reference.
- [ICD-10-PCS Code Request](02-ICD10_Code_Request.pdf) — Proposed code for AI sepsis tool.

**Synthetic Data**: [Data Synthesis.pdf](03-Data Synthesis.pdf) — 2,000 patients calibrated to CMS MedPAR FY2023.  
**CEA Model**: [markov_cea.pdf](04-markov_cea.pdf) — heemod with PSA.
