# NTAP with CEA markov model

## NTAP Application Package
Full materials for New Technology Add-on Payment submission to CMS, inspired by Yuan et al. (2020) on XGBoost for early sepsis diagnosis in ICUs.

- [Introduction & Overview](Introduction_and_Overview.pdf) — Study citation, synthetic data link, CEA reference.
- [ICD-10-PCS Code Request](ICD10_Code_Request.pdf) — Proposed code for AI sepsis tool.

**Synthetic Data**: [sepsis_ai_vs_sofa_synthetic_data.csv](sepsis_ai_vs_sofa_synthetic_data.csv) — 2,000 patients calibrated to CMS MedPAR FY2023.  
**CEA Model**: [markov_cea.Rmd](markov_cea.pdf) — heemod with PSA.
